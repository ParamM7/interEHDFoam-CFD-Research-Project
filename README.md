# interEHDFoam-CFD-Research-Project-MED-ITNU-27
This repository contains files and resources for our collaborative CFD research project.

Literature Naming Convention
All research papers stored in the literature folder must follow the naming rules below.

Create a folder for each topic
Example:
literature/
├── Turbulence/
├── Heat_Transfer/
└── Mesh_Generation/

File naming format
TXX_[Title of paper][First Author][Second Author or "et al"]

Year code rules

TXX → Year 20XX

Examples: 2025 → T25, 2021 → T21, 2010 → T10

NXX → Year 19XX

Examples: 1999 → N99, 1985 → N85

Examples
T25_LowReTurbulenceModeling_Smith_et al.pdf
T21_VortexSheddingSimulation_Jones_Williams.pdf
N98_LES_BackwardFacingStep_Taylor_et al.pdf
